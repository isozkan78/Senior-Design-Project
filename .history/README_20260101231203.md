# AI-Supported Forest Fire Risk Prediction System (Antalya)

## Project Overview
This project aims to develop a Deep Learning (LSTM) based model to predict forest fire risks in the Antalya region using satellite imagery. The system processes spatiotemporal data (MODIS, SRTM) via Google Earth Engine.

## Tech Stack
- **Platform:** Google Earth Engine (GEE)
- **Language:** Python 3.x
- **Libraries:** TensorFlow, Scikit-learn, GeoPandas, Earth Engine API

## Data Sources
1. **Dynamic:** MODIS Terra Vegetation Indices (MOD13A1) & Land Surface Temperature (MOD11A2)
2. **Static:** NASA SRTM Digital Elevation Model
3. **Target:** NASA FIRMS Active Fire Data
